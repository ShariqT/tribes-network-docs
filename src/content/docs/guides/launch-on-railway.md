# Launching your first Tribes server

Starting your first Tribes server is very easy to do. There is already an integration with the Railway platform. This is a cloud
hosting company started during the pandemic that has a very good 
reputation for easy of use and friendliness. 

You can create an account on [Railway.app](https://www.railway.app). After you have created this account, click on the following button to
create an instance of a Tribes server on your account. 


[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/deploy/tribes-server?referralCode=FR-b_V&utm_medium=integration&utm_source=template&utm_campaign=generic)
